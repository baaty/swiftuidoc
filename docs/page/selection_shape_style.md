### 説明

選択された要素の背景として使用できるスタイル

### 使い方

    SelectionShapeStyle(isSelected: 関連するビューが選択されているか(Bool) = true)

### 例

#### 基本的な使い方

    var selectionShapeStyle: SelectionShapeStyle

### 宣言

    struct SelectionShapeStyle
