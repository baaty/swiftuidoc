### 説明

子機を縦に並べその間に配置された仕切りでサイズを変更できるレイアウトコンテナ

### 使い方

    VSplitView {
        コンテンツ
    }

    VSplitView(content: コンテンツ)

### 例

#### 基本的な使い方

    VSplitView {
        // コンテンツ
    }

### 宣言

    struct VSplitView&lt;Content&gt; where Content : View
